This repository is a demonstration of using agentic AI tooling to find 
improved treatments of Chagas disease in low-resource settings.

# Contents
* `AGENTS.md` contains context for agents in this setting on Chagas disease including information on why this disease has not been a priority for drug discovery.
* `agents` contains a sample agent for research through design approaches to problem solving for hard to address goals.
* `skills` directory contains a sample skill for collaborative user persona elucidation and discovery.
